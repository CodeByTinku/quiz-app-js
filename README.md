# Quiz App (JavaScript)

A simple, browser-based quiz application built with HTML, CSS, and vanilla JavaScript.
This repository contains a lightweight multiple-choice quiz that tracks the user's score
and displays results at the end. It is designed for learning and easy customization.

## Features

- Multiple-choice questions with immediate feedback
- Score tracking and final result display
- Simple, dependency-free implementation
- Easy to extend with new questions or UI styles

## Installation

1. Clone the repository: `git clone https://github.com/CodeByTinku/quiz-app-js.git`
2. Navigate into the folder: `cd quiz-app-js`
3. Open `index.html` in any modern web browser

## Usage

- Click an answer to select it. The correct answer will be highlighted.
- Use the Next button to proceed to the following question.
- After the last question the app shows your total score and a Play Again option.

## Customization

- Edit `quiz.js` to add, remove, or correct questions and answers.
- You can randomize questions or add timers as enhancements.
